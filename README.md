# Handwritten Table to Excel Converter

A Python tool that extracts handwritten table data from images and converts it into structured Excel spreadsheets.

## Features
- Load image of handwritten table
- Crop and process image
- Extract text using OCR
- Convert table into structured data
- Export data to Excel (.xlsx)

## Tech Used
- Python
- OpenCV
- pytesseract (OCR)
- pandas
- openpyxl

## How to Run

```bash
pip install -r requirements.txt
python3 main.py
